# Sentinel Agent

Sentinel is a production-ready, AI-powered email security agent designed to automatically ingest, analyze, and respond to threats in real-time. 

Built for modern security operations, it features a beautifully designed liquid-glass dashboard for monitoring incidents, combined with a seamless Chrome Extension that allows users to instantly analyze actively opened emails.

## Key Features
- **Real-Time Extension:** One-click email analysis directly inside Gmail.
- **AI-Powered Pipeline (Qwen):** A robust 6-stage threat reasoning engine (Ingest, Scan, Analyze, Decide, Respond, Report).
- **Liquid Glass Dashboard:** A premium, real-time interface for incident monitoring and review.
- **Supabase Integration:** Persistent tracking of all security events and real-time status updates through SSE.

## Local Installation
1. Start the next server via `npm run dev` in the `/dashboard` directory.
2. Install the extension locally via the onboard zip located in the Dashboard UI.
